## 📌 About the Plugin
**Surge XT** is a synthesizer plugin designed for professional music producers using **FL Studio** and other DAWs. It delivers a wide range of sound design capabilities, from analog warmth to digital complexity, powered by advanced modulation and synthesis engines.

## ❓ What is Surge XT?
Surge XT combines subtractive, wavetable, and FM synthesis in one plugin. With powerful modulation routing, flexible filters, and hundreds of high-quality presets, it serves as a creative powerhouse for sound designers and composers working in FL Studio.

## ✅ Why Choose Surge XT?
- Fully open-source and actively maintained  
- Rich library of presets and sound banks  
- Multi-engine synthesis: FM, wavetable, and classic subtractive  
- Flexible modulation matrix and envelopes  
- Works perfectly inside FL Studio and other VST hosts  
- Lightweight, stable, and CPU-efficient  
- Supports MIDI automation and macro control  

---

## 🎚️ Key Features
- Up to 8 voices per patch with polyphonic modulation  
- 3 oscillators per voice with advanced routing  
- Dual filters with multiple models (ladder, SVF, comb, etc.)  
- Step sequencer, LFOs, and envelopes per layer  
- Integrated effects rack (reverb, delay, chorus, phaser, EQ)  
- Full MPE (MIDI Polyphonic Expression) support  
- Scalable user interface and custom themes  
- VST3 / CLAP plugin formats supported  

---

## 🖥️ System Requirements
- Windows 10/11 (64-bit)  
- 4 GB RAM (8 GB recommended)  
- FL Studio or compatible DAW (VST3 / CLAP host)  
- 250 MB free disk space  

## ▶️ Usage
1. Download and extract the **Surge XT Plugin** installer.  
2. Run the setup and select your FL Studio plugin directory.  
3. Open FL Studio → Add → More plugins → Refresh plugin list.  
4. Load **Surge XT** and start exploring presets or build your own sounds.
